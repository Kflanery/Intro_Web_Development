HTTP

Hypertext Transfer Protocol (HTTP) defines how browsers and servers exchange data on the web.

Client–Server Model

The client sends a request, and the server responds with data such as web pages or files.

URL

A Uniform Resource Locator (URL) identifies the location of a resource on the internet.

HTTP Headers

Headers carry metadata such as content type, caching rules, and authentication information.

HTTPS

HTTPS encrypts HTTP traffic using TLS to protect data from interception and tampering.

Multiplexing

HTTP/2 allows multiple requests and responses to be sent simultaneously over a single connection.

QUIC Protocol

QUIC is the transport protocol used by HTTP/3, improving speed, reliability, and reducing latency.
