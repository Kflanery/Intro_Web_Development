Introduction

Hypertext Transfer Protocol (HTTP) is one of the most important technologies behind the World Wide Web. Every time a user loads a webpage, streams a video, submits a form, or opens a cloud app, HTTP is involved in moving data between a client and a server. Even though most users never see it directly, HTTP is the backbone of modern web communication.

This website explains what HTTP is, how it works, how it has changed over time, and why new versions like HTTP/2 and HTTP/3 matter for performance and security.

Quick Summary

HTTP uses a request-response model. A browser asks for a resource, and the server returns the content and information about that content.
