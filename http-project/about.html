Student Introduction

My name is Kess Flanery, and this website was created for IT 3203 Web Development at Kennesaw State University. I am interested in technology and cybersecurity, and I chose HTTP because it is one of the most important protocols behind the modern web.

About This Project

The purpose of this project is to explain how HTTP works, how it has changed over time, and why newer versions such as HTTP/2 and HTTP/3 matter. The site was built with HTML and CSS using a clean, professional design.
